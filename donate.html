Donate to Code for DC

Thank you for your interest in helping to support Code for DC! Contributions of any size are helpful and welcome.

Donations to Code for DC are processed via Code for America. Code for America is a 501(c)(3) nonprofit and nonpartisan organization (EIN number 27-1067272), and all donations are tax-deductible to the fullest extent of U.S. law.

Important: In order for your donation to be sent to Code for DC, you MUST select “Code for DC” on the “If your donation is for a Brigade, please select the Brigade Name” drop-down menu. If you do not make that selection, your donation will not go to Code for DC.

All donations are unrestricted. If you are interested in becoming a sponsor for Code for DC, please send an email to team@codefordc.org.
